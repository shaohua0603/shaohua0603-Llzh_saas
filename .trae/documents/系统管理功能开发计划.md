# 系统管理功能开发计划

## 一、需求分析

### 1.1 系统管理功能模块（来自pc-feature-spec.md）

系统管理包含以下二级菜单：
- 企业介绍（企业文化介绍、岗位文化介绍）
- 部门管理
- 正式员工
- 岗位管理
- 规则配置
- 菜单配置
- 字典管理
- 系统参数
- 流程管理
- 流程配置
- 附件管理
- 打印管理（模版配置、打印配置）

### 1.2 UI设计规范（来自ui-design-spec.md）

- 统一页面布局：顶部导航栏60px、左侧菜单200px、四级导航栏40px
- 列表页面模板：搜索栏 → 工具栏 → 数据表格 → 分页
- 详情页面模板：信息分组展示、操作记录、审核记录
- 表单页面模板：字段分组、验证规则、联动逻辑
- 色彩系统、字体系统、间距布局规范
- 响应式设计规范

### 1.3 项目规则配置（来自project_rules.md）

- Vue 3 + TypeScript + Element Plus
- 组件开发规范：使用Composition API、<script setup>语法
- 命名规范：大驼峰组件名、小驼峰变量函数名
- 按钮操作规范：增改查跳转到新页面、删除弹窗二次确认
- 表格组件规范：排序、搜索、列宽调整、悬浮提示、文本截断、分页、多选、全选
- 日期时间选择组件规范：中文友好、年/月/日/时间选择
- 路由管理规范：懒加载、路由守卫、动态路由
- 状态管理规范：小/中/大型状态分别使用不同方案

## 三、开发规范要求

### 3.1 组件开发标准

- 使用Vue 3 Composition API + <script setup>语法
- 使用CommonTable组件实现列表页面
- 使用CommonForm组件实现表单页面
- 遵循按钮操作规范：增改查跳转到新页面、删除弹窗二次确认
- 支持表格排序、搜索、列宽调整、悬浮提示、文本截断
- 日期时间选择组件使用中文显示

### 3.2 页面设计标准

- 列表页面：搜索栏 → 功能按钮栏 → 表格 → 分页
- 详情页面：信息分组展示、审核记录时间线
- 表单页面：字段分组、验证规则、联动逻辑
- 四级导航栏：与菜单名称保持一致

### 3.3 数据权限

- 所有列表查询自动应用数据权限过滤
- 支持全部数据、本部门数据、本部门及以下、本人数据、自定义数据权限
