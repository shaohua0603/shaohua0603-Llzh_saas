企业介绍功能包含两个子功能模块：
1. **企业文化介绍**：展示和管理企业文化相关信息
2. **岗位文化介绍**：展示和管理各工厂岗位文化相关信息

## 三、功能需求详细分析

### 3.1 企业文化介绍功能

**数据字段**：
- 企业文化标题（必填，最大100字符）
- 企业简称（必填，最大50字符）
- 企业文化内容（必填，富文本）

**功能特性**：
- 单条记录管理（每个企业只有一条企业文化介绍）
- 支持富文本编辑
- 支持图片上传
- 支持内容预览
- 支持保存草稿
- 支持发布/取消发布

**权限控制**：
- 平台管理员：可查看和编辑所有企业的企业文化
- 劳务公司管理员：可查看和编辑本公司的企业文化
- 工厂管理员：可查看所属劳务公司的企业文化

### 3.2 岗位文化介绍功能

**数据字段**：
- 工厂（必填，下拉选择）
- 岗位文化标题（必填，最大100字符）
- 岗位文化内容（必填，富文本）

**功能特性**：
- 多条记录管理（每个工厂可以有多条岗位文化介绍）
- 支持富文本编辑
- 支持图片上传
- 支持内容预览
- 支持保存草稿
- 支持发布/取消发布
- 支持按工厂筛选

**权限控制**：
- 平台管理员：可查看和管理所有工厂的岗位文化
- 劳务公司管理员：可查看和管理本公司合作工厂的岗位文化
- 工厂管理员：可查看和管理本工厂的岗位文化

<template>
  <div class="company-culture-container">
    <el-card class="form-card">
      <template #header>
        <div class="card-header">

        :model="formData"
        :rules="formRules"
        label-width="140px"
        label-position="right"
      >

          <el-input
            v-model="formData.title"
            placeholder="请输入企业文化标题"
            maxlength="100"
            show-word-limit
          />
        </el-form-item>
        
        <el-form-item label="企业简称" prop="shortName">
          <el-input
            v-model="formData.shortName"
            placeholder="请输入企业简称"
            maxlength="50"
            show-word-limit
          />
        </el-form-item>
        
        <el-form-item label="企业文化内容" prop="content">
          <RichTextEditor
            v-model="formData.content"
            placeholder="请输入企业文化内容"
            :height="400"
            :auto-save="true"

        </el-form-item>

<script setup lang="ts">
import { ref, reactive, onMounted } from 'vue'
import { ElMessage } from 'element-plus'
import RichTextEditor from '@/components/RichTextEditor.vue'

const formRef = ref()
const saving = ref(false)

const formData = reactive({
  title: '',
  shortName: '',
  content: ''
})

const formRules = {
  title: [
    { required: true, message: '请输入企业文化标题', trigger: 'blur' }
  ],
  shortName: [
    { required: true, message: '请输入企业简称', trigger: 'blur' }
  ],
  content: [
    { required: true, message: '请输入企业文化内容', trigger: 'blur' }
  ]
}

const fetchData = async () => {
  try {
    const response = await fetchCompanyCulture()
    if (response.code === 200) {
      Object.assign(formData, response.data)
    }
  } catch (error) {
    ElMessage.error('获取数据失败')
  }
}

const handleSave = async () => {
  if (!formRef.value) return
  
  await formRef.value.validate(async (valid: boolean) => {
    if (valid) {
      saving.value = true
      try {
        const response = await saveCompanyCulture(formData)
        if (response.code === 200) {
          ElMessage.success('保存成功')
        } else {
          ElMessage.error(response.message || '保存失败')
        }
      } catch (error) {
        ElMessage.error('保存失败')
      } finally {
        saving.value = false
      }
    }
  })
}

onMounted(() => {
  fetchData()
})
</script>
