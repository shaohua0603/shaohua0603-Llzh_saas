# 蓝领智汇原型设计开发文档

## 1. 项目概述

蓝领智汇是一个面向劳务工人、劳务公司、工厂和平台管理员的SaaS平台，旨在解决劳务工人招聘、管理、培训、薪酬、福利等问题，实现劳务工人与工厂的双向沟通，提高劳务工人工作效率，降低劳务工人劳动强度。

本原型设计基于Vue 3 + TypeScript + Element Plus技术栈，实现了多角色、多终端的响应式设计，所有数据均使用Mock.js进行模拟，无需后端支持即可展示完整的业务流程。

## 4. 功能实现

### 4.1 工人端（移动端）

- **注册登录**：手机号注册、手机号登录、忘记密码
- **用工招聘**：厂家直招、平台招聘、劳务招聘
- **衣食住行**：嵌入第三方H5商城页面
- **技能提升**：技能开发、教育、职业规划
- **消息**：个人待办、消息通知
- **我的**：岗前培训、考勤、请假、离职、调岗、特殊情况、投诉/建议、个人信息、工资条、文娱活动、我的社团、工作转介绍、个人简历、员工福利

### 4.2 劳务公司端

- **PC端**：招聘管理、工人管理、考勤管理、结算管理、部门管理、角色管理
- **移动端**：工人管理、考勤管理、个人中心

### 4.3 工厂端

- **PC端**：招聘管理、工人管理、岗前培训、结算管理、部门管理、角色管理
- **移动端**：工人管理、岗前培训、个人中心

### 4.4 平台管理员端

- **租户管理**：劳务公司、工厂、直营劳务管理
- **套餐配置**：劳务公司套餐、工厂套餐、直营劳务套餐
- **招聘管理**：查看工厂发给平台代为招聘的需求清单
- **注册用户查询**：查看所有在移动端注册的用户
- **空闲工人管理**：查看未归属劳务公司和未分配工厂的工人

## 6. 移动端和PC端的设计差异

### 6.1 布局差异

| 终端 | 布局类型 | 特点 |
|------|----------|------|
| 移动端 | 卡片式布局 + 底部导航栏 | 单栏布局，内容垂直排列，操作简单直观，适合小屏幕设备 |
| PC端 | 左侧菜单 + 右侧内容区域 | 多栏布局，信息密度高，操作便捷，适合大屏幕设备 |

### 6.2 组件差异

| 终端 | 组件选择 | 特点 |
|------|----------|------|
| 移动端 | 小尺寸按钮、卡片、列表 | 触摸友好，间距较大，字体清晰 |
| PC端 | 标准尺寸按钮、表格、表单 | 鼠标操作友好，信息展示全面，交互丰富 |

### 6.3 交互差异

| 终端 | 交互方式 | 特点 |
|------|----------|------|
| 移动端 | 点击、滑动、手势 | 操作简单直接，反馈及时 |
| PC端 | 点击、拖拽、键盘操作 | 操作精准高效，支持复杂交互 |

## 11. 总结

本原型设计基于Vue 3 + TypeScript + Element Plus技术栈，成功实现了蓝领智汇平台的核心功能，包括工人端、劳务公司端、工厂端和平台管理员端的完整界面。通过Mock.js模拟数据，无需后端支持即可展示完整的业务流程，为后续的正式开发奠定了坚实的基础。

原型设计充分考虑了不同角色的使用场景和设备类型，实现了响应式设计，确保在移动端和PC端都能提供良好的用户体验。同时，通过规范的代码结构和设计理念，为项目的可持续发展提供了保障。

后续开发可以在此原型的基础上，逐步实现真实的后端API接口，完善功能细节，优化性能和安全，最终将蓝领智汇平台打造成一个真正实用的SaaS解决方案。
