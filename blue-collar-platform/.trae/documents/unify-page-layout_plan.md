## 三、统一页面布局规范

### 1. 布局结构

参考工作转介绍页面，统一的页面布局应包含以下区域：

1. **查询条件区域**：
   - 包含各种筛选条件输入框、下拉选择器、日期选择器等
   - 使用 Element Plus 的 el-form 组件，inline 模式
   - 包含搜索和重置按钮

2. **功能按钮区域**：
   - 包含新增、导出、导入等功能按钮
   - 位于查询条件区域下方
   - 使用 flex 布局，右对齐

3. **数据统计区域**：
   - 以提示性问题的方式展示统计信息
   - 不需要卡片图标，直接以文字形式展示
   - 位于功能按钮区域下方

4. **列表展示区域**：
   - 使用统一的表格组件
   - 包含分页、排序、筛选等功能
   - 位于数据统计区域下方

### 2. 样式规范

- **容器样式**：
  - 页面容器使用白色背景，有适当的内边距
  - 各区域之间有适当的间距

- **查询条件区域**：
  - 使用 el-form 组件，inline 模式
  - 表单元素之间有适当的间距

- **功能按钮区域**：
  - 使用 flex 布局，右对齐
  - 按钮之间有适当的间距

- **数据统计区域**：
  - 使用简洁的文字展示
  - 与其他区域有明显的区分

- **列表展示区域**：
  - 使用统一的表格组件样式
  - 表格行高、列宽等保持一致

### 3. 实现示例

参考工作转介绍页面的布局实现，确保所有页面都遵循相同的结构和样式。
