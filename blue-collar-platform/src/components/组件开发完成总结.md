## 开发规范遵循

两个组件都严格遵循了《蓝领智汇平台开发规范》:

1. **代码风格规范**
   - 使用2个空格缩进
   - 使用单引号或反引号
   - 每行代码长度不超过120个字符
   - 使用中文注释

2. **组件开发规范**
   - 使用Vue 3 Composition API
   - 使用`<script setup>`语法
   - 组件结构清晰，逻辑分离
   - 遵循单一职责原则
   - 组件可复用性设计

3. **样式管理规范**
   - 使用scoped样式
   - 避免使用!important
   - 响应式设计适配不同屏幕
   - 移动端、平板、桌面端适配

4. **TypeScript使用规范**
   - 为所有变量添加类型
   - 为函数参数和返回值添加类型
   - 使用接口定义复杂类型
   - 集中管理类型定义

5. **命名规范**
   - 组件文件使用大驼峰命名法
   - 变量名使用小驼峰命名法
   - 函数名使用小驼峰命名法
   - 常量使用全大写字母
