# 合同管理模块开发总结

## 模块概述

合同管理模块是蓝领智汇 SaaS 系统中合同管理功能的核心模块，用于管理劳务公司与工人之间的劳动合同。该模块包含三个主要页面，提供了完整的合同管理功能。

## 开发时间

2026-02-26

## 技术栈

- Vue 3 + TypeScript
- Composition API + `<script setup>` 语法
- Element Plus UI 组件库
- 通用组件（CommonTable、CommonForm、PersonSelect、RichTextEditor、FileUpload）

## 文件结构

```
src/views/labor-company/contract/
├── Contract.vue          # 合同列表页
├── ContractForm.vue      # 合同新增/编辑页
└── ContractDetail.vue    # 合同详情页
```

## 功能说明

### 1. 合同列表页（Contract.vue）

**功能特性：**
- 使用 CommonTable 组件展示合同列表
- 支持多条件筛选（合同状态、结算方式、签订日期范围）
- 支持全局搜索
- 支持表格排序（点击表头排序）
- 支持分页功能
- 支持自定义列表和列设置
- 支持导出功能

**数据字段：**
- 合同编号
- 甲方
- 乙方
- 结算方式
- 合同状态
- 合同签订日期
- 合同生效日期
- 合同过期日期
- 合同金额

**操作功能：**
- 新增合同：跳转到新增页面
- 查看详情：跳转到详情页面
- 编辑合同：跳转到编辑页面（仅未签订和签订中状态）
- 删除合同：弹窗二次确认（仅未签订和签订中状态）

**状态管理：**
- 未签订（UNSIGNED）：info 标签
- 签订中（SIGNING）：warning 标签
- 已签订（SIGNED）：success 标签
- 已取消（CANCELLED）：danger 标签

### 2. 合同新增/编辑页（ContractForm.vue）

**功能特性：**
- 使用 CommonForm 组件构建表单
- 集成 PersonSelect 组件选择乙方（工人）
- 集成 RichTextEditor 组件编辑合同内容
- 集成 FileUpload 组件上传合同附件
- 支持表单验证
- 支持保存草稿功能
- 支持自动保存草稿（30秒间隔）

**表单字段：**
- 合同编号（必填，自动生成）
- 甲方（必填）
- 乙方（必填，人员选择器）
- 结算方式（必填，下拉选择：月结/日结/周结）
- 合同状态（必填，下拉选择：未签订/签订中/已签订/已取消）
- 合同签订日期（必填，日期选择器）
- 合同生效日期（必填，日期选择器）
- 合同过期日期（必填，日期选择器）
- 合同金额（必填，数字输入）
- 合同内容（必填，富文本编辑器）
- 合同附件（可选，文件上传）

**表单验证：**
- 所有必填字段验证
- 合同金额数值验证
- 日期逻辑验证（生效日期不能早于签订日期）

### 3. 合同详情页（ContractDetail.vue）

**功能特性：**
- 信息分组展示（基本信息、合同内容、合同附件、操作记录）
- 使用 el-descriptions 组件展示基本信息
- 使用 el-timeline 组件展示操作记录
- 支持合同附件预览和下载
- 支持打印功能
- 支持编辑和删除操作（仅未签订和签订中状态）

**信息分组：**

1. **基本信息**
   - 合同编号
   - 结算方式
   - 甲方
   - 乙方
   - 合同签订日期
   - 合同生效日期
   - 合同过期日期
   - 合同金额

2. **合同内容**
   - 富文本内容展示
   - 支持HTML格式

3. **合同附件**
   - 文件列表展示
   - 支持预览（图片、PDF）
   - 支持下载

4. **操作记录**
   - 时间线展示
   - 操作时间
   - 操作人
   - 操作内容
   - 备注信息

**操作功能：**
- 编辑：跳转到编辑页面
- 删除：弹窗二次确认
- 打印：调用浏览器打印功能
- 预览附件：弹窗预览
- 下载附件：直接下载

## 数据接口说明

### 1. 获取合同列表

```typescript
GET /api/v1/contracts
```

**请求参数：**
- page: 当前页
- pageSize: 每页条数
- sortField: 排序字段
- sortOrder: 排序方式（ascending/descending）
- keyword: 搜索关键词
- status: 合同状态
- settlementMethod: 结算方式
- signDateStart: 签订日期开始
- signDateEnd: 签订日期结束

**响应数据：**
```typescript
{
  code: 200,
  message: 'success',
  data: {
    list: Contract[],
    total: number,
    page: number,
    pageSize: number
  },
  timestamp: number
}
```

### 2. 获取合同详情

```typescript
GET /api/v1/contracts/:id
```

**响应数据：**
```typescript
{
  code: 200,
  message: 'success',
  data: Contract,
  timestamp: number
}
```

### 3. 创建合同

```typescript
POST /api/v1/contracts
```

**请求数据：**
```typescript
{
  contractNo: string,
  partyA: string,
  partyBId: string,
  partyBName: string,
  settlementMethod: string,
  status: string,
  signDate: string,
  effectiveDate: string,
  expiryDate: string,
  amount: number,
  contractContent: string,
  attachments: string[]
}
```

### 4. 更新合同

```typescript
PUT /api/v1/contracts/:id
```

**请求数据：**同创建合同

### 5. 删除合同

```typescript
DELETE /api/v1/contracts/:id
```

### 6. 导出合同

```typescript
GET /api/v1/contracts/export
```

**请求参数：**同获取合同列表

## 类型定义

```typescript
// 合同数据类型
interface Contract {
  id: string
  contractNo: string
  partyA: string
  partyB: {
    id: string
    name: string
    phone: string
    department: string
  }
  settlementMethod: 'MONTHLY' | 'DAILY' | 'WEEKLY'
  status: 'UNSIGNED' | 'SIGNING' | 'SIGNED' | 'CANCELLED'
  signDate: string
  effectiveDate: string
  expiryDate: string
  amount: number
  contractContent: string
  attachments: File[]
  createdAt: string
  updatedAt: string
}

// 合同状态枚举
enum ContractStatus {
  UNSIGNED = 'UNSIGNED',      // 未签订
  SIGNING = 'SIGNING',        // 签订中
  SIGNED = 'SIGNED',          // 已签订
  CANCELLED = 'CANCELLED'     // 已取消
}

// 结算方式枚举
enum SettlementMethod {
  MONTHLY = 'MONTHLY',        // 月结
  DAILY = 'DAILY',            // 日结
  WEEKLY = 'WEEKLY'           // 周结
}
```

## 响应式设计

所有页面均支持响应式设计，适配不同屏幕尺寸：

- **桌面端（> 1024px）**：多列布局，完整功能展示
- **平板端（768px - 1024px）**：双列布局，优化触控和鼠标操作
- **移动端（< 768px）**：单列布局，触控优化

## 注意事项

1. **合同编号生成**：新增合同时自动生成，格式为 `HT{年月}{随机4位}`
2. **日期验证**：合同生效日期不能早于签订日期，合同过期日期必须晚于生效日期
3. **状态限制**：已签订和已取消的合同不能编辑和删除
4. **附件限制**：单个文件大小不超过 50MB，最多上传 10 个文件
5. **草稿保存**：表单支持自动保存草稿，间隔 30 秒
6. **数据权限**：根据用户岗位配置的数据权限过滤合同数据

## 开发规范遵循

本模块严格遵循《蓝领智汇平台开发规范》：

1. **代码风格**
   - 使用 2 个空格缩进
   - 每行代码长度不超过 120 个字符
   - 使用单引号或反引号
   - 使用中文注释

2. **组件设计**
   - 遵循单一职责原则
   - 组件可复用性设计
   - 组件通信使用 props 和 emit

3. **按钮操作规范**
   - 增改查操作：点击按钮后跳转到新的页面进行操作
   - 删除操作：在当前页面以弹窗形式进行二次确认

4. **样式管理**
   - 使用 scoped 样式
   - 避免使用 !important
   - 响应式设计适配不同屏幕

5. **类型定义**
   - 使用 TypeScript 进行类型定义
   - 为所有变量添加类型
   - 使用接口定义复杂类型
